✨ Features

⚡

6 Tab Interface

Commands, History, Guide, Files, Settings, and System Monitor tabs organized for maximum productivity and ease of use.

📊

System Monitor

Real-time CPU, Memory, Disk, and Network monitoring with color-coded usage indicators. Always know your system status.

🎨

8 Beautiful Themes

From catboy to ocean, forest to neon, choose from 8 stunning color schemes with customizable accent colors and backgrounds.

🔍

Smart Search

Filter commands and files in real-time. Find what you need instantly with arrow key navigation.

📋

Clipboard Integration

Paste with Ctrl+V or Shift+Insert. Works seamlessly with your clipboard for quick command entry.

💻

Multi-Language Support

Execute Node.js, Python, and PowerShell commands directly. Perfect for developers working with multiple languages.

📦 Installation

🚀 Automatic

Download and run the installer:

build.bat
install.bat

🔧 Manual

Build from source:

cargo build --release
copy target\release\nyashell.exe %APPDATA%\Local\NyaShell\
powershell -ExecutionPolicy Bypass -File install-profile.ps1
